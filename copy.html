5 Projetos para incluir no Portfolio (Fonte:FreeCodeCamp)

  1. Portfolio Pessoal
  2. Product Landing Page - OK
  3. Site Clone: Recriar um site existente do zero. Ex: Google. - OK
  4. Um App CRUD: Que a pessoa pode criar, ler, atualizar e deletar. Ex: To-Do List ou Notes. (usar React) - OK
  5. Site conectado com API: communicação com backend de algum servidor externo. Ex.: Weather API ou RESTful API
  6. A Perfect score Website

Não precisa ser projetos todos separados. Pode criar um CRUD que usa API para obter dados de um arquivo JSON
